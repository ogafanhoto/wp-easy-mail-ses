Easy to use. Send emails via AWS Simple Email Service(SES)

== Description ==

Amazon SESを簡単にメールを送れます。
このプラグインはwp_mail() 関数を上書きし、オプションページを作ります。

注意: Amazon SESを使ってメールを送る前に、あなたのメールアドレス（あるいはドメイン）を認証している必要があります。参照 http://docs.aws.amazon.com/ja_jp/ses/latest/DeveloperGuide/verify-addresses-and-domains.html

機能:

* AWSのアクセスキーやシークレットキー、送信者Emailアドレスなどを設定できます。
* 設定された認証情報やEmailアドレスが正いか検証します。

ロードマップ:

* テストメッセージの送信
* 送信した情報や制限の表示

== Installation ==

1. プラグインをダウンロードします。
2. `/wp-contents/plugins/` ディレクトリにアップロードします。
3. ワードプレスのメニューから有効化します。
